The Database Object

(since Version 0.0.1)
Database -- This Object represents a Database (a Folder)

Description:

Before you can do something with a Table, you need an instance of a Database Object.

With this instance you then can execute SQL Statements, use the function executeQuery() to do this.

Example:

$db = new Database("MyDatabase");

$resultSet = $db->executeQuery("SELECT nr, name, prename FROM person WHERE nr=1");
while($resultSet->next()) {
	echo "<br>" . $resultSet->getCurrentValueByName("nr") . ": " . $resultSet->getCurrentValueByName("name") . "<br>";
}
