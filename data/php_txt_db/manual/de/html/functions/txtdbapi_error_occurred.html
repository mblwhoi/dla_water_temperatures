txtdbapi_error_occurred

(seit Version 0.2.0)
txtdbapi_error_occurred -- Gibt true zurück falls im letzten executeQuery() Aufruf ein Fehler auftrat

Beschreibung:

bool txtdbapi_error_occurred ()

Gibt true zurück falls im letzten executeQuery() Aufruf ein Fehler auftrat.
Falls kein Fehler auftrat leifert die Funktion false.

Rückgabewert:

true falls im letzten executeQuery() Aufruf ein Fehler auftrat, sonst false
