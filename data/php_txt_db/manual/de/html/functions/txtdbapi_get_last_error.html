txtdbapi_get_last_error

(seit Version 0.2.0)
txtdbapi_get_last_error -- Gibt den letzten Fehler zurück und entfernt ihn vom Fehler-Stack

Beschreibung:

string txtdbapi_get_last_error ()

Gibt den letzten Fehler als String zurück und entfernt ihn vom Fehler-Stack, d.h. der nächste
txtdbapi_get_last_error() aufruf liefert den nächsten Fehler.

Rückgabewert:

den letzten Fehler als String
