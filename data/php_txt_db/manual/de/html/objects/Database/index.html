Das Database Objekt

(seit Version 0.0.1)
Database -- Diese Objekt stellt eine Datenbank (in einem Ordner) dar

Beschreibung:

Befor man irgendetwas auf einer Tabelle machen kann, benötigt man eine Instanz des
Database Objektes.

Von diesem Objekt aus kann man dann SQL aufrufe absetzten.
Dafür steht die Funktion executeQuery() zur Verfügung.

 

Beispiel:

$db = new Database("DatenbankName");

$resultSet = $db->executeQuery("SELECT nr, name, vorname FROM personen WHERE nr=1");
while($resultSet->next()) {
	echo "<br>" . $resultSet->getCurrentValueByName("nr") . ": " . $resultSet->getCurrentValueByName("name") . "<br>";
}
