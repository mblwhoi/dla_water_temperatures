getLastInsertId

(seit Version 0.2.0)
getLastInsertId -- Gibt die Insert ID zurück

Beschreibung:

int getLastInsertId ()

Gibt den Wert der ersten inc Spalte zurück, für den letzten Datensatz welcher bei einem INSERT Statement angefügt wurde.
(Dazu muss also erst mal ein INSERT auf der verwendeten Database Instanz ausgeführt worden sein!)

Rückgabewert:

letzte Insert ID
