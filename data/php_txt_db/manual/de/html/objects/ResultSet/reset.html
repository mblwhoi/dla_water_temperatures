reset

(seit Version 0.0.1)
reset -- Setzt die Position des ResultSet's vor den ersten Datensatz

Beschreibung:

reset ()

Diese Funktion setzt die Position des ResultSet's vor den ersten Datensatz.
Wieso vor den Ersten ? Damit man mit dem nächsten next() Aufruf zum ersten
Datensatz gelangt. So wird zB folgender Quelltext möglich:

$resultSet->reset();
while($resultSet->next()) {
	echo "Aktuelle Reihe: ";
	$arr=$resultSet->getCurrentValues();
	print_r($arr);
}
