getColumnNames

(seit Version 0.0.1)
getColumnNames -- Gibt ein Array zurück, welches die Spaltennamen des ResultSet's enthält.

Beschreibung:

array getColumnNames ()

Gibt ein Array zurück, welches die Spaltennamen des ResultSet's enthält. Diese könnten zB.
später bei getCurrentValueByName() oder getValueByName() verwendet werden.

Rückgabewert:

Ein Array welches die Spaltennamen enthält.
