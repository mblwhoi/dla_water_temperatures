Das ResultSet Objekt

(seit Version 0.0.1)
ResultSet -- Diese Objekt stellt das Ergebnis einer SELECT Abfrage dar

Beschreibung:

Nachdem man mit Database->executeQuery() eine SELECT Abfrage ausgeführt hat
bekommt man ein ResultSet Objekt zurück, welches die Daten dieser Abfrage enthält.
Mit dem ResultSet Objekt kann man dann durch die Datensätze des Ergebnisses navigieren
und diese abfragen.

Beispiel:

$db = new Database("DatenbankName");

$resultSet = $db->executeQuery("SELECT nr, name, vorname FROM personen WHERE nr=1");
while($resultSet->next()) {
	echo "<br>" . $resultSet->getCurrentValueByName("nr") . ": " . $resultSet->getCurrentValueByName("name") . "<br>";
}
