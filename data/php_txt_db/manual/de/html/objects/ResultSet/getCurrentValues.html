getCurrentValues

(seit Version 0.0.1)
getCurrentValues -- Gibt die Werte der aktuellen Reihe als Array zurück.

Beschreibung:

array getCurrentValues ()

Gibt die Werte des aktuellen Datensatzes als array zurück. Die werte sind im Array
gleich angeordnet wir in der Sql-Abfrage. Der aktuelle Datensatz, bwz. die aktuelle Reihe kann zB. mit den
Funktionen next(), prev() oder setPos() verändert werden.

Hier ist eine Verwendung von list() sehr gut geeignet, z.B.

$db=new Database("MeineDatenbank");

$rs=$db->executeQuery("SELECT nr, name, ort, email FROM Personen");
while($rs->next()) {
	list($nr, $name, $ort, $mail)=$rs->getCurrentValues();
	echo "$nr: $name aus $ort (Email: $mail)";
}

Rückgabewert:

Die Werte der akuellen Reihe als Array;
