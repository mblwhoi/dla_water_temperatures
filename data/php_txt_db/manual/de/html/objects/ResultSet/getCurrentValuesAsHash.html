getCurrentValuesAsHash

(seit Version 0.1.3-alpha5)
getCurrentValuesAsHash -- Gibt die Werte der aktuellen Reihe als Hash Array zurück.

Beschreibung:

array getCurrentValuesAsHash ()

Gibt die Werte des aktuellen Datensatzes als hash array zurück. Die werte sind im Array
gleich angeordnet wir in der Sql-Abfrage. Der aktuelle Datensatz, bwz. die aktuelle Reihe kann zB. mit den
Funktionen next(), prev() oder setPos() verändert werden.

Da es ein hash array ist können mit den spalten-namen die werte ausgelesen werden: 

$db=new Database("TestDB");

$rs=$db->executeQuery("SELECT * FROM woerter GROUP BY wort, art ORDER BY wort DESC LIMIT 0,3");
while($rs->next()) {
	$arr=$rs->getCurrentValuesAsHash();
	echo $arr['wort'] . $arr['art'] . "<br>";
}

Rückgabewert:

Die Werte der akuellen Reihe als Hash Array
